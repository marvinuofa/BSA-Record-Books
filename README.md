# BSA Record Books

A web application for managing and tracking BSA record books.

## Tech Stack

* React
* TypeScript
* Vite
* TailwindCSS
* Supabase
* TanStack Query
* React Router

## Installation

Clone the repository:

```
git clone https://github.com/marvinuofa/BSA-Record-Books.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
